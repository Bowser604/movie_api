Movie API

Information about movies, directors and genres provided.
sign up, update, remove and create a list of their favorite movie information.

Endpoint URL HTTP Method Query Parameters Request data Response data
Return a list of all movies to user /movies GET None Array of JSON objects holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET None A JSON object holding data about a single movie containing title, description, genre, year and director. Example: { Title: "Gladiator", Description: "A former Roman General sets out to exact vengeance against the corrupt emperor.", Genre: { Name: "Action/Drama", year: "2000", }, Director: { Name: "Ridley Scott", Bio: 'Described by film producer Michael Deeley as "the very best eye in the business".', Birth: "November 30, 1937", } }
Return data about a movie genre /movies/genre/[genreName] GET None Array of JSON objects holding data about all movies by genre requested
Return data about a director by name /movies/director/[directorName] GET None Array of JSON objects holding data about all movies by director requested
Register new user /users POST A JSON object holding data about user to add. Example: { 'Name': 'John Doe',
'Username': '[username]',
'Email': [email@example.com],
'Birth-date': '1987-08-01' } 		A JSON objects holding data about the user that was added, with assigned i.d.: { '_id': '*****',
'Name': 'John Doe',
'Username': '[username]',
'Email': [email@example.com],
'Birth-date': '1987-08-01' }
Update user info /users/[id] PUT A JSON object holding data about the user that needs to be updated. Example: { "name": "Mary", "favoriteMovies": [] } A JSON object holding data about the updated user information. Example: { "id": 2, "name": "Mary", "favoriteMovies": [] }
Allow users to add a movie to their list of favorites /users/[username]/[movieID] PUT None A message showing text that a movie has been added.
Allow users to delete a movie from their list of favorites /users/[username]/[movieID] DELETE None A message showing text that a movie has been removed.
Deregister existing user info /users/[username]/[movieid] DELETE None A message showing text that a movie has been removed.
