Movie API

Update user info /user/:Username PUT A JSON object holding data about the user that needs to be updated. Example: { acknowleaged: true, insertedId: null, matchedCount: 1, modifiedCount: 1, userredcount: 0 }
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieId POST A JSON object holding data about the user that needs to be updated. Example: { _id: ObjectId('65efbacb30d32340e8b98953'), user_firstname: 'Bob'. user_lastname: 'Wilson', email: 'bob.wilson@example.com', password: 'bobpass', birthday: ISODate ('1988-11-30T00:00:00.000z'), user_movies: [], FavoriteMovies: [] } A JSON object holding data about the updated user information. Deregister existing user info /users/:Username DELETE None A message showing text that a user has been removed.
